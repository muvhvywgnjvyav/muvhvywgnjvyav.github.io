ヒューマンメイド 【 HUMANMADE 】とは
ヒューマン メイド ( HUMAN MADE )は、 ア ベイシング エイプ 【 A BATHING APE ( BAPE ) APE ベイプ エイプ アベイシングエイプ 】の創業者 NIGO 氏がデザインするブランドでありプロジェクト。「過去と未来の融合」をテーマに2010年からスタート。
HUMAN MADE は NIGO 氏が所有する莫大な ヴィンテージ アーカイブからインスパイアを受け、それをモダンに表現したコレクションが特徴的です。
2020年に オフホワイト ( Off-White )の ヴァージル・アブロー 氏と ルイ・ ヴィトン ( LOUIS VUITTON ) とコラボ。
2021年9月に NIGO 氏は KENZO ( ケンゾー )のアーティスティック・ディレクターに就任。
2022年2月に 高橋盾 氏【 ジョニオ JONIO 】の アンダーカバー ( UNDERCOVER ) と 藤原ヒロシ （ フラグメント fragment ） 氏から受け継いだ LAST ORGY 2（ ラストオージー 2 LASTORGY ）を復活させた。
グラフィックデザイナー VERDY （ ヴェルディ ）氏の手掛ける Girls Don’t Cry ( ガールズドントクライ GDC ガルドン )とのコラボや KAWS 氏（ カウズ ）、 フューチュラ （ FUTURA ）、 ダブルタップス （ WTAPS ）、 アディダス ( adidas )、リーバイス ( LEVI’S )、 木梨サイクル 、 ディズニー （ Disney ）、 スヌーピー （ SNOOPY PEANUTS ピーナッツ ）、 カリーアップ （ CURRY UP ）等とのコラボを行っている。
多くの著名人【 PHARRELL WILLIAMS ( ファレル )、 木村拓哉 ( キムタク )、那須川天心 】等が愛用している。
