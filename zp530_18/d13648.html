POC の Tectal MTB ヘルメットは、アグレッシブなトレイルやエンデューロ MTB ライダーのニーズを満たせるよう開発されました。 最先端の保護力、強化された気流、そして洗練された北欧デザインが合わさったトレイルの覇者のようなヘルメットです。

ターンでタイヤのグリップを試すようなハードなライド、なるべく遠く、高くを目指すジャンプ、または挑戦の多いライドを走る場合、クラッシュはほぼ回避能です。 結局、ライダーは、転びながら上達するのです。 そして、転倒したライダーを守るのが、Tectal MTB ヘルメットです。

特徴
EPS ( 発泡スチロール ) ライナー : EPS ライナーは最適化された保護を提供します。
ユニボディシェル構造 : ユニボディポリカーボネートシェルはヘルメットの構造的完全性を高めます。
拡張された保護ゾーン : こめかみや後頭部のカバー力を高めるデザイン
調整可能な 360° フィットシステム
調節可能なピーク - 雨や風からの保護のために調節されています
風洞試験で最適化された通気
アイガレージ : サングラスを後部の通気口にしっかりと保管します
Recco リフレクター : 問題が発生した場合、救助サービスが簡単かつ迅速にあなたの位置を特定するのに役立ちます
重量 : 340g (M)
